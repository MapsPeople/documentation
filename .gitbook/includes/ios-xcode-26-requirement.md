---
title: iOS Xcode 26 requirement
---

{% hint style="info" %}
From version 4.17.0 (the first minor release after April 28, 2026) MapsIndoors requires a minimum iOS version of 16.6 and Xcode 26 as [Apple will require apps to be built with Xcode 26](https://developer.apple.com/news/upcoming-requirements/?id=02032026a) to be distributed via the App Store.
{% endhint %}
